# Clinical Signal Workbench

Clinical Signal Intelligence repository for exploratory analysis of CDISC ADaM datasets.

## Planned Features

- Longitudinal laboratory and vital signs analytics
- Trend detection
- Peak and nadir detection
- Change-point analysis
- Composite organ scores
- Biomarker network analysis
- Patient phenotyping
- Interactive Shiny dashboard
